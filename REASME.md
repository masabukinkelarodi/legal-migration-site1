# Legal Migration International - Landing Page

Premium, high-converting bilingual landing page designed for **Legal Migration International / SAMI**. 
Features instant dynamic client-side English/Russian translation and active cross-platform social media lead generation funnels (WhatsApp, TikTok, Facebook, Instagram).

## 🚀 Features
- **Bilingual Core Architecture:** Instant toggle switch between English and Russian without page reloads.
- **Social Lead Generation Hub:** Integrated WhatsApp click-to-chat action triggers alongside structured multi-channel community touchpoints.
- **Responsive Fluid UI:** Programmed completely using Tailwind CSS utility architectures.
- **Iconography Suite:** Leverages FontAwesome 6 CDN injections for maximum visual delivery.

## 🛠️ Technical Stack
- HTML5
- Tailwind CSS (via CDN configuration)
- Native Vanilla JavaScript ES6 (Translation Engine & UI controls)

## 📁 Repository Structure
- `index.html` - Primary landing logic and language engine.
- `assets/` - Folder reserved for hosting local brand elements (`logo-drc.jpg`, `logo-sami.jpg`).

## ⚙️ Installation & Deployment
1. Clone or download this repository.
2. Ensure your custom assets are placed inside the `assets/` subdirectory.
3. Open `index.html` directly in any standard modern web browser or deploy directly to production solutions like GitHub Pages, Vercel, or Netlify.
